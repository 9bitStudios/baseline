Baseline UI

Baseline UI is an, er, baseline framework for your web and mobile projects. It provides a number of basic and sensible defaults for a number of common elements. It does not try to be the entirety of your UI. You could also import other UI frameworks into your project as well with minimal collision.

Modal Dialogs

Modal dialogs can be easily initialized in the following manner...

var modal = new Baseline.Components.Modal('#myModalSelector')

You will want to give the constructor a selector for your modal dialog. The dialog can then be called like so...

modal.show()
